Here’s everything you asked for, battle-tested and ready to drop in.

The tmux config: design philosophy is Alt-key bindings for everything you do constantly — no prefix needed. Alt+h/j/k/l for pane navigation. Alt+v for voice-to-text toggle. Alt+z for zoom. Prefix + | for vertical split, prefix + - for horizontal.


The voice-to-text setup has five pre-solved gotchas. The main one: tmux’s run-shell reaps all child processes when the script exits. Fix is setsid, which creates a new session leader that survives parent exit. The other one worth knowing: debounce must use lockfile age, not wav file age — the wav file’s mtime updates continuously as arecord writes, so its age is always zero. Using wav age as the debounce blocks all stop attempts permanently.


whisper.cpp: build from source with SDL2 enabled. base.en model is 142MB and transcribes at 15x real-time on CPU. Plenty fast for dictation. We also grabbed large-v3 for batch work, but base.en is the one for the live toggle.

"The whole thing took about an hour of iteration but should take you 10 minutes with the bugs pre-squashed."
