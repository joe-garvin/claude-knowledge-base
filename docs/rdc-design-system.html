Rules
  • Always sentence case for headings — never title case or all caps (labels excepted)

  • Do not set body copy in condensed or decorative variants

Guidelines

Usage rules

These rules maintain brand integrity across every touchpoint — decks, social, partner materials, and beyond.

Logo
Do

Use approved Box files. Maintain clear space equal to the "r" height on all sides. Match logo variant to background color.

Don't

Recreate the logo or alter its proportions. Add drop shadows or effects. Place the full-color logo on complex photographic backgrounds without a protective field.

Color
Do

Lead with Coral for CTAs and primary actions. Use Squid to anchor dark surfaces. Use the Wine-to-Coral gradient for decorative accents only.

Don't

Use Mist as a primary page background — it's an accent color. Set Coral text on Wine (insufficient contrast). Introduce off-brand colors without approval from the design lead.

Typography
Do

Use Suisse Int'l for all branded communications. Use sentence case for headings. Create hierarchy through weight variation.

Don't

Use more than 3 weights on a single page. Use condensed or decorative variants for body copy. Italicize headings as an emphasis style.

Photography & illustration

RDC uses a combination of custom illustrations and editorial photography. Photography should feel candid, human, and professional — not stock-photo staged. Illustrations use the brand palette and a geometric line style. Custom illustration files are on file in Box under Custom RDC illustrations/.
