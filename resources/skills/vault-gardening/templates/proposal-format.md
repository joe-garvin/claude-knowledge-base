# Per-note proposal template

Use this format for every proposal in Step 3. Show it to Joe and wait for a response before
doing anything.

---
**Note:** `[path/to/note.md]`

**Proposed tags:** `[tag1]`, `[tag2]`
*(Current tags: [existing tags, or "none"])*

**Proposed related notes:**
- [[note-title]]
- [[note-title]]

Type **y** to apply, **e** to edit before applying, **s** to skip, or **q** to quit the session.

---

## Notes on filling it in

- Omit the "Proposed tags" block if the note already has adequate tags
- Omit the "Proposed related notes" block if the note already has a full `## Related notes` section
- If proposing both tags and links, include both blocks
- Keep proposed related notes to 3–5 maximum — most thematically proximate first
