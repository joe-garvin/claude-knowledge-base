# RDC company context

General-purpose context documents on Red Door Collaborative (RDC), intended to be shared with LLMs and AI agents (Claude Desktop, Claude Code) at the start of any project that involves RDC work.

## When to use

Reference these documents when starting a new project, task, or conversation that involves RDC clients, deliverables, team coordination, or internal operations. Paste the relevant file(s) into the conversation context, or point the agent to this folder.

## Files

| File | Contents |
|------|----------|
| [rdc-overview.md](rdc-overview.md) | Company background, mission, positioning, brand pillars, and tone of voice |
| [rdc-org-and-team.md](rdc-org-and-team.md) | Full org chart with names, roles, and reporting structure across all departments |
| [rdc-services-and-clients.md](rdc-services-and-clients.md) | Service model, service categories, current and past clients, and deliverable types |
| [rdc-operations-and-pain-points.md](rdc-operations-and-pain-points.md) | How work flows, tools and platforms, SharePoint structure, and known operational pain points |
