# Project Notes and Context

This folder contains project notes and context documentation for work and personal initiatives in progress.

## Subfolders

- **`work/`** — Client and professional projects. Includes briefs, workflows, deliverables, and Claude conversation outputs (e.g., Azure SMB campaign briefs, Databricks blog workflows, RDC internal strategies).
- **`personal/`** — Personal projects and initiatives, documented with the same approach.

## About

Each project gets its own subfolder. Within each you'll find notes, context, process documentation, and outputs that chronicle how the project evolved — useful for reference, iteration, and onboarding to similar work later.
